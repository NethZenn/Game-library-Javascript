# Game-library-JavaScript
A simple game store project built with JavaScript.  
This project is made to practice JavaScript fundamentals like arrays, objects, DOM manipulation, and functions.

## Features

- Display game information
- Show game ratings
- Filter games by rating
- Filter games by performance (high-end / low-end)
- Calculate total game prices
- Calculate total storage size
- Simulate game downloading
- Simulate launching a game

## Technologies Used

- HTML
- CSS
- JavaScript

## JavaScript Concepts Practiced

- Variables
- Functions
- Objects
- Arrays
- forEach()
- map()
- filter()
- reduce()
- DOM manipulation
- Event handling
- setTimeout()
